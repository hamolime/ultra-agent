# 🤖 الـ Agent الخارق (Ultra Agent Bot)

بوت تليجرام ذكي جداً ومساعد شخصي متكامل مبني بلغة Python، ومبرمج بلهجة مصرية ودودة ومحترفة. البوت بيشتغل بعقل نماذج Llama القوية عن طريق Groq API، وبيقدّم خدمات ذكية زي الرد على الرسائل، تحليل الصور، والجدولة والتذكير.

## ✨ الميزات الأساسية:
- **🤖 ذكاء اصطناعي خارق:** شات متكامل ومستقر باستخدام موديل `llama-3.3-70b-versatile` للرد على كل الأسئلة والاستفسارات.
- **👁️ تحليل الصور (Vision):** ابعت له أي صورة وهيحللها ويشرحهالك بالتفصيل بفضل موديل `llama-3.2-11b-vision-preview`.
- **⏰ منبه وجدولة مهام:** نظام جدولة داخلي متطور بيفكرك بمواعيدك ومهامك في الوقت اللي تحدده.
- **🔒 أمان كامل:** البوت مقفول ومخصص لصاحب البوت بس عن طريق الـ `ADMIN_CHAT_ID`.

## 🛠️ طريقة الاستخدام جوه تليجرام:
- **للدردشة العادية:** اكتب أي سؤال أو رسالة والبوت هيرد عليك فوراً.
- **لتحليل الصور:** ارفع أي صورة في الشات والبوت هيقوم بالواجب.
- **لجدولة مهمة:** اكتبها بالفورمات ده:
  `جدول: اسم المهمة | YYYY-MM-DD HH:MM:SS`
  *مثال:* `جدول: ميعاد الكورس | 2026-06-15 18:00:00`

## 🚀 التشغيل والـ Deployment:
المشروع جاهز ومكتوب بملف الـ `Dockerfile` عشان يشتغل علطول على منصات زي **Railway** أو **Render**.

### المتغيرات المطلوبة (Environment Variables):
- `TELEGRAM_TOKEN`: توكن البوت من BotFather.
- `GROQ_API_KEY`: مفتاح الـ API من منصة Groq Cloud.
- `ADMIN_CHAT_ID`: الرقم التعريفي لحسابك على تليجرام عشان البوت يشتغل ليك لوحدك.
